# Basketball Fan Engagement App

This application provides a tool for basketball fans and tournament hosts to monitor teams, players, and even users. It allows for transparency from league operations, allowing for public track of games, matches, and statistics. Additionally it allows basketball fans to show their engagement and likes. An excel sheet is not enough for this app because of limitations in scalability, poor automation, and the poor user interface as compared to a specialized app.
